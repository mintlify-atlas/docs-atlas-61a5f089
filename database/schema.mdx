  <Accordion title="v_coder_risk_analysis">
    **Purpose:** Risk analysis view used by Team Leaders to identify at-risk coders. Compares soft skills scores against academic performance and provides both the stored risk level and an AI-calculated risk classification.

    ```sql
    COMMENT ON VIEW v_coder_risk_analysis IS 'Análisis de riesgo de cada coder para el TL';
    ```

    **Columns returned:**

    | Column | Source | Description |
    |---|---|---|
    | `id` | `users.id` | Coder ID |
    | `email` | `users.email` | Coder email |
    | `full_name` | `users.full_name` | Coder display name |
    | `autonomy` | `soft_skills_assessment.autonomy` | Autonomy score |
    | `time_management` | `soft_skills_assessment.time_management` | Time management score |
    | `average_score` | `moodle_progress.average_score` | Module average score |
    | `current_risk_level` | `risk_flags.risk_level` | Most recent unresolved risk flag level (`low` if none) |
    | `calculated_risk_level` | computed | Rule-based risk: `high` if autonomy ≤ 2 AND score < 70; `medium` if either condition; `low` otherwise |
    | `total_activities` | computed | Count of distinct activities |
    | `completed_activities` | computed | Count of completed activities |

    **Filter:** `WHERE users.role = 'coder'` and only unresolved risk flags (`rf.resolved = FALSE`).

    **Risk calculation logic:**
    ```sql
    CASE
      WHEN ssa.autonomy <= 2 AND mp.average_score < 70 THEN 'high'
      WHEN ssa.autonomy <= 2 OR  mp.average_score < 70 THEN 'medium'
      ELSE 'low'
    END AS calculated_risk_level
    ```
  </Accordion>
</AccordionGroup>
